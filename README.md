### SEELE-UI是一套基于React和TypeScript、使用函数式编程的桌面端开源UI库

#### 安装
SEELE-UI可以通过npm安装[npm package](https://www.npmjs.com/package/seele-ui):

```$ npm install --save seele-ui```
